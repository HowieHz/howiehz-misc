# Graphwar 杀手

[English](../../en/tools/graphwar-killer/) | **简体中文**

Graphwar 杀手用于从 [Graphwar](https://graphwar.com/graphwar_1/index.html) 截图标定坐标、点选路径，并生成可直接输入 Graphwar 的双绝对值函数或阶跃函数表达式。
